How to Use SG CourseTracker



Home Page


Select your A-Level grade for each subject in the select boxes. Click one of the three buttons, NUS, NTU or SMU, to be directed to its respective page where your Rank Points and a table of courses you are eligible to apply for in that University will be displayed.



There is a navigation bar at the top where you can go to the web administrator page.





Web Admin Page


CRUD operations are available for all three Universities where you can retrieve existing data from the database by pressing the 'Retrieve' button, create a new record by pressing the 'Insert' button, edit existing records by pressing the 'Update' button or delete a record by pressing the 'Delete' button.



Each record should be referenced by its ID (which can be found through retrieval) when performing CRUD operations.
